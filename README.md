## 🚀 Terminal Cheat Sheet

To compile and run any code file from my terminal:

```bash
# 1. Go into the folder
cd FolderName

# 2. Compile the C++ file
clang++ -std=c++17 FileName.cpp -o run

# 3. Run it
./run

# 4. Clean up the generated executable
rm run
